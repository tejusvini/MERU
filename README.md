# MERU

MERU is a structural pattern engine and pattern programming language.

The goal of MERU is to represent cognition, structure, and generative systems using compact pattern expressions.

MERU explores:

- Structural pattern languages
- Generative laws
- Context-based pattern resolution
- Pattern memory and recognition
- Pattern-driven computation

## Repository Structure

/spec
Formal definition of the MERU language.

 /examples
Example pattern programs.

 /docs
Conceptual explanations and research notes.

 /tools
Parsers, interpreters, or utilities.
